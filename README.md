# MedCura – Health Analysis System

## Overview

MedCura is a desktop health analysis application developed using Python and Tkinter. The system allows users to describe their symptoms and automatically detects possible health conditions based on the input.

The application analyzes symptoms, provides possible diagnoses, and suggests basic medications.

This project demonstrates how natural symptom detection and rule-based diagnosis systems can be implemented using Python.

## Features

* Symptom detection from user input text
* Interactive GUI built using Tkinter
* Automatic diagnosis suggestions
* Medication recommendations
* Detailed information about symptoms
* Scrollable UI for large medical data
* Modern interface design

## Technologies Used

* Python
* Tkinter (GUI)
* Rule-based symptom detection
* Medical data mapping

## How the System Works

1. User enters symptoms in the text box.
2. The system detects symptoms using keyword matching.
3. Additional questions appear based on detected symptoms.
4. The system generates a diagnosis summary.
5. Suggested medications are displayed.
